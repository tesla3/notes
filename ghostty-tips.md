# Ghostty Default Keybindings

Notation: lowercase = bare key, `Shift` written explicitly when needed.

| Action         | macOS              | Linux                          |
| -------------- | ------------------ | ------------------------------ |
| New tab        | `Cmd+t`            | `Ctrl+Shift+t`                 |
| Previous tab   | `Cmd+Shift+[`      | `Ctrl+Shift+Left`              |
| Next tab       | `Cmd+Shift+]`      | `Ctrl+Shift+Right`             |
| Previous tab   | `Cmd+Shift+Tab`    | `Ctrl+Shift+Tab`               |
| Next tab       |                    | `Ctrl+Tab`                     |
| Go to tab 1–10 | `Cmd+1` … `Cmd+0`  | `Alt+1` … `Alt+0`              |
| Paste          | `Cmd+v`            | `Ctrl+Shift+v`                 |
| Copy           | `Cmd+c`            | `Ctrl+Shift+c`                 |
| Close tab      | `Cmd+w`            | `Ctrl+Shift+w`                 |
| New window     | `Cmd+n`            | `Ctrl+Shift+n`                 |

> **Pattern:** macOS `Cmd` → Linux `Ctrl+Shift` (mostly).
