# MIT BWSI "Girls who Can" Fall Saturday Program

Source: https://bwsi.mit.edu/saturday-programs/  
Date: 2026-05-01

## Summary

The Fall "Girls who Can" Saturday program is **open registration, not selective**. Any eligible female student can enroll — it's not a competitive merit-based admission process like the BWSI Summer program.

## Key Evidence

1. **No competitive application process** — Unlike the BWSI Summer program (which requires prerequisite courses, benchmark exams, essays, recommendations, and has competitive admission), the Saturday programs use simple enrollment/registration.

2. **Language used**: Website says program is "open to motivated students" and uses "register" rather than "apply" — indicating open enrollment vs. selective admission.

3. **Historical precedent**: A 2023 announcement stated "Must fill out application link to be considered" with first spots going to 10th/11th graders, but 9th graders can apply too — suggesting first-come, first-served with priority tiers, not merit-based selection.

4. **"Space is limited"** — Spring program notes encourage applying "early to secure your spot" — implying capacity-based limits, not selectivity.

## Eligibility Requirements (Fall 2026)

- Residing in the US and attending high school physically in the US
- Currently enrolled in **10th or 11th grade**
- While open to students of any gender identity, the program is **designed to encourage and support female students in STEM**
- Completely **free**
- Virtual format (via Zoom)

## Timeline

- Fall 2026 program announced in **August 2026**
- Registration opens shortly after announcement
- Program runs ~8 Saturdays, typically October–December
- Sessions: Saturdays 11am-2pm EST

## Courses Offered

- Program Autonomous Cars
- Program Cognitive Assistants
- Applied Engineering, AI & Future Technologies
- Design, Build and Fly Model Aircraft
- Program Racing Drones
- Learn Python
- Digital Logic & Computer Design
- Learn BioTech
- Computational Problem Solving

## Contrast with BWSI Summer Program

The Summer program is **highly selective** (competitive admission):
- Multi-step application: register for online prerequisites → benchmark exam → full application
- Requires essays, teacher recommendation, grades/test scores
- For rising seniors (11th graders entering 12th grade)
- Some courses in-person at MIT, others virtual
- $2,400 fee for families >$200k income; free for qualifying families

The Saturday programs are introductory/feeder programs designed to build skills and interest before applying to the competitive summer institute.
